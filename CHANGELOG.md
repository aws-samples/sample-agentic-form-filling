# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial release of airline-checkin-agent
- AWS Bedrock-powered AI agent for airline check-in operations
- Automated passenger check-in functionality
- Flight information retrieval
- Customer service interactions using generative AI
- Integration with Amazon Bedrock AgentCore
- CloudFront distribution for secure content delivery
- Secrets Manager integration for credential management
- CloudWatch logging and X-Ray tracing for observability

### Security
- Implemented least privilege IAM policies
- Enabled encryption at rest and in transit
- Added input validation and sanitization
- Configured secure session management

### Added
- Initial project structure

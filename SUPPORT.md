### Reporting Issues

If you encounter bugs or have feature requests:

1. Check existing [GitHub Issues](../../issues) to see if your issue has been reported
2. If not, create a new issue with:
   - Clear description of the problem or feature request
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Environment details (AWS region, Python version, etc.)

## Scope of Support

This is sample code provided for demonstration and educational purposes. Support is provided on a best-effort basis through GitHub issues.

### What We Support

- Questions about the sample code and its functionality
- Bug reports with reproducible steps
- Feature requests and enhancements
- Documentation improvements

### What We Don't Support

- Production deployment assistance
- Custom feature development
- Third-party integrations not included in this sample
- General AWS account or billing questions
